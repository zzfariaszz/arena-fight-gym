# Arena Fight Gym

Página institucional de academia de luta marcial — MMA, Muay Thai, Jiu-Jitsu e Boxe.

Desenvolvido com Vite + React e deploy automatizado via GitHub Actions.

## Tecnologias

- React 18 + Vite
- ESLint
- Vitest
- GitHub Actions (CI/CD)
- GitHub Pages

## Rodando localmente

```bash
npm install
npm run dev
```

## Pipeline CI/CD

- **CI:** roda lint, testes e build a cada push/PR na branch `main`
- **Deploy:** publicação automática no GitHub Pages a cada push na `main`
